# 活动群运营助手 H5 网页

一个功能完整的活动群运营管理工具,帮助您轻松管理群内活动的各个环节。

## 🎯 主要功能

### 1. 📝 活动通知文案生成
- 输入活动项目名称、时间、地点、路程等信息
- 自动生成规范的活动通知文案
- 包含完整的活动信息和注意事项
- 一键复制分享到群聊

### 2. 📊 活动参与情况统计
- 实时统计参与人数(已确认/待确认/无法参加)
- 饼图展示参与状态分布
- 折线图展示参与趋势
- 自动计算确认率和响应率

### 3. 📷 活动相册管理
- 支持批量上传活动照片
- 网格布局展示照片
- 点击照片可放大预览
- 照片本地存储

### 4. 📧 活动复盘邮件生成
- 自动汇总活动数据
- 填写活动亮点和改进建议
- 生成格式规范的复盘邮件
- 一键复制发送

## 🚀 使用方法

### 快速开始
1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖,开箱即用
3. 所有数据保存在浏览器本地存储

### 操作流程
1. **生成通知** → 填写活动信息 → 点击生成 → 复制文案
2. **统计数据** → 输入参与人数 → 查看可视化图表
3. **上传相册** → 选择照片 → 批量上传 → 点击预览
4. **生成复盘** → 填写总结信息 → 生成邮件 → 复制发送

## 🔧 技术栈

- **HTML5** - 语义化标签,现代化结构
- **CSS3** - Flexbox/Grid布局,动画效果
- **JavaScript (ES6+)** - 原生JS实现所有功能
- **Chart.js** - 数据可视化图表库

## 📝 数据存储

所有数据使用浏览器的 `localStorage` 存储,包括:
- 活动基本信息
- 参与人数统计
- 上传的照片(限制20张)

**注意**: 清除浏览器数据会导致信息丢失,请及时复制保存重要内容。
